# Dutch Language Pack for MikoPBX

[![MikoPBX](https://img.shields.io/badge/MikoPBX-2025.1.1+-blue.svg)](https://github.com/mikopbx/Core)
[![License](https://img.shields.io/badge/License-GPLv3-green.svg)](LICENSE)

Complete Dutch language support for MikoPBX including 638 professional voice prompts and full UI translation.

## Installation

1. Open **System → Extension Modules → Marketplace**
2. Search for "Dutch Language Pack"
3. Click **Install** and **Enable**
4. Go to **General Settings** and select **Nederlands (nl-nl)**

## What's Included

- 638 Dutch voice files (greetings, menus, digits, letters)
- 16 translation files for complete UI localization
- Automatic integration with MikoPBX
